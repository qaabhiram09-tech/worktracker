# WorkTracker – Freelance CRM

Your personal freelance project tracker for web development work.

## Setup

1. Extract this folder to `D:\abhiram\worktracker`
2. Open terminal (Command Prompt or PowerShell) in that folder
3. Run:

```
npm install
npm run dev
```

4. Open your browser at: **http://localhost:5173**

That's it — the app runs locally, data saves automatically in your browser.

## Build for production (optional)

```
npm run build
npm run preview
```
